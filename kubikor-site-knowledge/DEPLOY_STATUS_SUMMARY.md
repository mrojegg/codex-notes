## Как обновлять сайт дальше
1. Отредактировать файл: `index.html`.
2. Закоммитить изменения:
   - `git add index.html`
   - `git commit -m "Update site"`
3. Отправить в GitHub:
   - `git push`
4. Дождаться зелёного workflow в Actions (деплой пройдет автоматически).

## Зафиксированное правило (безопасный путь)
Все будущие правки на сайте делаем только через безопасный процесс:
1. Создать отдельную ветку от `main`.
2. Внести изменения и проверить локально.
3. Сделать commit в этой ветке.
4. Смёржить ветку в `main` только после проверки.
5. Публикация в прод происходит только из `main` через GitHub Actions.
